✨ Prompt Enhancer 测试页面

测试步骤：

  1. 确保插件已在 chrome://extensions/ 中启用
  2. 点击插件的刷新按钮 🔄
  3. 刷新此页面 Cmd+R
  4. 打开开发者工具 Cmd+Option+I → Console
  5. 查看是否有 🚀 Prompt Enhancer: Content script loaded
  6. 点击下方输入框，查看是否出现紫色边框和 ✨ 按钮
这是一个 contenteditable 元素，用于测试富文本输入框
调试信息：

如果控制台没有任何 "Prompt Enhancer" 日志，说明 content script 未加载。

请检查 chrome://extensions/ 页面：

  • 插件是否显示"错误"按钮？点击查看错误信息
  • 插件详情页 → 是否有权限问题？
