> 心有灵犀一点通

一键优化你在 AI 输入框中的提示词，再发送。支持 ChatGPT、Claude、Gemini、DeepSeek 等 50+ 平台。安装即用，免费额度无需配置。

## 安装

- **Chrome / Edge**：商店上架后提供链接。
- **从源码**：`git clone https://github.com/gentlemouse/prompt-enhancer.git` → `npm install` → `npm run build` → 在 `chrome://extensions/` 中加载 `dist` 目录。

## 使用

- 快捷键：`Cmd+Shift+E`（Mac）/ `Ctrl+Shift+E`（Windows/Linux）。
- 免费额度用完后，在扩展设置中配置自己的 API Key 即可无限使用。
