> 心有灵犀一点通

Browser extension that optimizes your prompt before sending it to AI. Works on ChatGPT, Claude, Gemini, DeepSeek and 50+ platforms. Install and use — no config required for free tier.

## Install

- **Chrome / Edge**: Coming soon (store listing).
- **From source**: `git clone https://github.com/gentlemouse/prompt-enhancer.git` → `npm install` → `npm run build` → load the `dist` folder in `chrome://extensions/`.

## Usage

- Shortcut: `Cmd+Shift+E` (Mac) / `Ctrl+Shift+E` (Windows/Linux).
- Free tier: 10 uses; then set your own API Key in the extension for unlimited use.
